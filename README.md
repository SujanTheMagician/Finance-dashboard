# FinanceOS — Personal Finance Dashboard

> Built by **Sujan Anandh** · [GitHub](https://github.com/SujanTheMagician)

A full-featured personal finance web app with animated dark UI, built with React + Vite + Recharts.

## Features

- 🎬 **Animated splash screen** with typewriter effect & floating particles
- 📋 **4-step onboarding** — set up profile, income, budgets, goals
- 📊 **Dashboard** — KPI cards, area charts, donut chart, smart insights
- 💳 **Transactions** — add, filter, search, delete with category tagging
- 📈 **Analytics** — bar charts, daily trends, savings rate
- 🎯 **Budgets** — per-category tracking with over-budget alerts
- ⭐ **Goals** — financial milestones with progress bars
- 🌙 **Dark UI** with glass morphism, gradient text, micro-animations
- 📱 **Mobile responsive** sidebar with overlay

## Tech Stack

- React 19 + Vite
- Recharts (charts)
- React Router DOM
- CSS custom properties + keyframe animations (no external UI lib)
- LocalStorage persistence

## Getting Started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```
